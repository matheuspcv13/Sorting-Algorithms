# Sorting Algorithms

Este repositório foi criado para um trabalho prático de Sistemas de Informação na disciplina de LAB II. Ele contém implementações de vários algoritmos de ordenação em C++.

## Algoritmos Implementados

- Bubble Sort
- Merge Sort
- Quick Sort

## Estrutura do Projeto

Os arquivos de saída (txt) estão na pasta `output`, juntamente com o executável, para manter a organização no mesmo diretório.
